Generate idol-specific achievement questions to assess what the user has already accomplished.

These questions will be used to compare the user's achievements to the idol's milestones.

INPUT:
- idol_name: "{idol_name}"
- idol_profile_json:
{idol_profile_json}
- milestones_json (key milestones to compare against):
{milestones_json}
- user_age: {user_age}
- limit: {limit}

GOAL:
Generate {limit} achievement-focused questions that:
- Map directly to the idol's milestone categories (career, learning, finance, impact, mindset)
- Ask about concrete, verifiable accomplishments (not feelings or plans)
- Cover the breadth of the idol's journey up to the user's current age
- Allow the system to later do a 1:1 comparison between user achievements and idol milestones

IMPORTANT:
- Questions must be answerable with specific facts (titles, numbers, dates).
- Use simple language suitable for a mobile UI.
- Each question should target a different milestone category.
- Include at least one question about: career milestones, skills/learning, and a concrete project or creation.
- Do NOT ask for sensitive personal data.

OUTPUT JSON ONLY:

{
  "questions": [
    {
      "id": "ach_1",
      "title": "Short achievement area title",
      "prompt": "Specific question about what the user has accomplished in this area",
      "type": "multiline",
      "required": true,
      "options": null,
      "placeholder": "e.g. I built a ..., I achieved ...",
      "validation": { "min_len": 0, "max_len": 1000 },
      "category": "achievements",
      "mapping_hint": "user_achievement.career"
    }
  ]
}

RULES:
- Generate exactly {limit} questions.
- All questions should have category "achievements".
- mapping_hint should indicate the milestone category being targeted (e.g., "user_achievement.career", "user_achievement.learning").
- Prefer "multiline" type for open-ended achievement descriptions.
- Use "single_choice" or "scale" only when asking about levels or quantities.
- Frame questions in a way that even users with no achievements can answer honestly (e.g., "What have you built or created so far, if anything?").
